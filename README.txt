3. Project Folder description

4. Files:
	1. TestingModule.py
		- Module for testing phase
	2. TrainingModule.py
		- Module for training phase
	3. SaveFeatures.py
		- Module for saving extracted features to a file
	4. FeatureCalculation.py
		- Module for calculating distances between features
	5. Main.py
		- Entry point for the Face Recognition application
	6. haarcascade_eye.xml
		- Haar cascade file for eyes
	7. haarcascade_mcs_mouth.xml
		- Haar cascade file for mouth
	8. haarcascade_mcs_nose.xml
		- Haar cascade file for nose
	9. haarcascade_frontalface_default.xml
		- Haar cascade file for front file
	10. faces.zip
		- Dataset for our project
		

5. Steps to run a program:
	1. install following packages
		a. openCV
		b. numpy
		c. shutil
		d. collections
	3. Extract/Unzip the dataset
	2. Run Main.py
		python Main.py
